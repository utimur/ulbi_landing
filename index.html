Мои навыки

React - это декларативная, эффективная и гибкая библиотека JavaScript для создания пользовательских интерфейсов (UI). Она позволяет вам создавать сложные UI из небольших и изолированных частей кода, называемых «компонентами».

Redux — библиотека управления состоянием для приложений, написанных на JavaScript. Она помогает писать приложения, которые ведут себя стабильно/предсказуемо, работают на разных окружениях (клиент/сервер/нативный код) и легко тестируемы.

JavaScript - язык сценариев, с его помощью можно создавать интерактивные html-документы, производить вычисления, выполнять проверку допустимости данных без обращения к серверу.

Java — это язык программирования общего назначения, который следует парадигме объектно-ориентированного программирования и подходу «Написать один раз и использовать везде» . Java используется для настольных, сетевых, мобильных и корпоративных приложений.

Формальный язык описания внешнего вида документа, написанного с использованием языка разметки. Преимущественно используется как средство описания, оформления внешнего вида веб-страниц, написанных с помощью языков разметки HTML

Spring Framework обеспечивает комплексную модель разработки и конфигурации для современных бизнес-приложений на Java - на любых платформах. Ключевой элемент Spring - поддержка инфраструктуры на уровне приложения

Высокоуровневый язык программирования общего назначения, ориентированный на повышение производительности разработчика и читаемости кода. Синтаксис ядра Python минималистичен. В то же время стандартная библиотека включает большой объём полезных функций.

Фреймворк для создания веб-приложений на языке программирования Python, использующий набор инструментов Werkzeug, а также шаблонизатор Jinja2. Относится к категории так называемых микрофреймворков - минималистичных каркасов веб-приложений

Семейство Unix-подобных операционных систем на базе ядра Linux, включающих тот или иной набор утилит и программ проекта GNU, и, возможно, другие компоненты

Проекты, над которыми я работал
Home.io - платформа для интернета вещей

Интернет вещей – это полностью автоматизированный цикл работы приборов и систем за счет их подключения к беспроводной сети. Простыми словами: понятие интернета вещей предполагает взаимодействие по схеме «машина – машина» с минимальным участием человека.

Однако для разработки в этой области программисту необходимо обладать большим количеством знаний из разных сфер (Разработка клиент-серверной части, сетевые технологии, программирование микроконтроллеров). Идея концепции «платформа для интернета вещей Home.io» такова, что программисту, для того чтобы войти в мир интернета вещей, необходимо знать лишь язык программирования Python на базовом уровне.

LunGO - мобильное приложение "Мессенджер"

Приложение даёт возможность пользователю работать с сообщениями, работать с контактами, создавать групповые чаты, получать уведомления о новом сообщении, изменять личные данные.

Плюсы программы: интуитивно понятный интерфейс; кэширование основной информации, в результате чего уменьшается использование трафика; озвучивание текстовых сообщение для пользователей, нуждающихся в этом; использование быстрой базы данных, многопользовательские чаты. Минусы: необходимость наличия постоянного интернета.
